# Отчёт заказчика (Smeg report.docx) — статус

Документ от заказчика с скриншотами. Зафиксировано 2026-07-01.

## Реализовано

| # | Замечание | Решение |
|---|-----------|---------|
| 1 | HBAC11 — группировка по цветам | `model_group = HBAC11` + группировка аксессуаров включена |
| 2 | Фильтр «На главную» | `is_featured` в карточке товара + SKU в Контент → Главная |
| 3 | Аксессуары не работают | `resolveAccessoryCategoryRaw()` — матч по RU/EN/slug |
| 4 | Нет категорий в меню (вытяжки, смесители…) | Расширены `LARGE_FAMILIES` / `SMALL_FAMILIES` |
| 5 | Меню как smeg.com (Breakfast, Coffee…) | **Контент → Категории → Меню каталога** + шаблон smeg.com |
| 6 | Нет кнопки «Сбросить» после выбора цвета | Кнопка в режиме выбора цвета модели |
| 7 | D&G только по-русски | `name_en = Refrigerator of Art` (миграция + админка) |
| 8 | Цифры 76 / 50+ / 11 | Редактируются в **Контент → История бренда** |
| 9 | Matt чайники отдельно от gloss | `KLF03-MATT` / `KLF03-GLOSS` model_group |

## Админка: подгруппы меню

**Контент → вкладка Главная → блок «Категории»** (внизу):

- **Меню каталога (подгруппы)** — создать/редактировать группы
- В группу можно добавить:
  - **Категория** — ссылка в каталог
  - **Family** — линейка (Kettles, Hood…)
  - **Model group** — группа моделей (HBAC11…)
  - **SKU** — конкретный артикул

Кнопка **«Загрузить шаблон smeg.com»** — Breakfast, Coffee, Water и т.д.
